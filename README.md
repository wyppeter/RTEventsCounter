# RTEventsCounter

A central computational pipeline for identifying reverse transcription(RT)-associated terminations, mutations, and indels from RNA sequencing to e.g. analyse RNA structure probing data. This pipeline was developed by Peter Y. Wang (@wyppeter) in Matthew Simon’s lab at Yale University. For more details on its original application, refer to the associated paper at: 

Sexton et al., 2017 http://dx.doi.org/10.1021/acs.biochem.7b00323
